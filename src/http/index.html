Optical Character Recognition

About

My name is Nimrod Yahal, and this is my finals project for my cyber highschool major.
Optical Character Recognition (or OCR), is the process in which a program recognizes text in images. The uses for such a program are numerous - from digitizing old, printed documents to translation programs. It is one subcategory of 'Computer Vision', a field of study all about programming computers to process visual input - recognize text, faces, cars or lanes, etc.
Such a program is difficult to create the old-fashioned way, so instead a different was developed: Machine Learning.
The main difference between a program working by Machine Learning and one working by "the old fashioned way" (also called Knowledge Engineering) is that the first isn't programmed to solve the problem - it is programmed to learn how to solve the problem.
